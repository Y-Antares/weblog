chrome connector:

使用 shift++，可以展开文献列表，使用 - 可以折叠文献列表。

D/zotero/locate/engines.json已替换为这位博主的。

插件：

如何将插件导入到 Zotero 中呢？

依次点击：工具 -> 插件，进入到插件管理界面，然后将下载下来的插件(.xpi)直接拖入其中，或者在插件管理界面，点击右上角的设置按钮，选择 Install Add-on From File。将插件导入后，需要重启软件才会生效。插件导入后，下载下来的插件文件就可以删除了
